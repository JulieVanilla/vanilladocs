## Function: `{searchbox}`

```smarty
{searchbox placeholder="string"}
```

Writes the search box to the page.

### Parameters

Parameter       | Type      | Default   | Description
---             | ---       | ---       | ---
`placeholder`   | `string`  | Search    | Optional placeholder to use
